技術構成

4つのノイズレイヤーを合成して雨音を生成しています。
① ベース雨音: ブラウニアンノイズ → lowpass (800Hz) + highpass (100Hz) → LFO揺らぎ
② 雨粒ディテール: ホワイトノイズ → bandpass (2500Hz, Q=0.8) → LFO揺らぎ
③ 水しぶき: ホワイトノイズ → highpass (5000Hz) + lowpass (12000Hz) → LFO揺らぎ
④ 雨粒パチパチ: 短いノイズバースト(5〜30ms) → bandpass (ランダム 2000〜6000Hz) × エンベロープ減衰 → ランダム間隔で発火
①〜③は連続ノイズ+LFO揺らぎ、④は個別の雨粒インパクトを離散的にスケジューリングしています。
⑤ 遠雷: ブラウニアンノイズ(1〜3秒) → lowpass (150Hz) + エンベロープ(attack/decay) → 10〜60秒間隔でランダム発火
